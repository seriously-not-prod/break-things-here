## Overview

This application helps organize and manage festival events, providing tools for event planning, coordination, and execution.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Build

```bash
npm run build
```
